Download PyNIO

Precompiled PyNIO binaries are available only for a limited number of platforms. If you cannot find a matching binary here, you may find one at ESG. Alternatively, you can install ccplot >= 1.5, which no longer depends on PyNIO.

a i686 are basically all desktop 32-bit Intel and AMD platforms. Download even if you have a modern 64-bit hardware platform, but run a 32-bit operating system on it. x86-64 (also called AMD64, x64, EM64T or Intel 64) are modern 64-bit platforms such as Intel Core and AMD Athlon 64 or AMD Phenom (II). Download only if you run a 64-bit OS.

b Normally, you would like to download PyNIO compiled with the newest version of gcc.
